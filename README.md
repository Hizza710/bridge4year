# 課題名
NEW YEAR BRIDGE - Let's Reflect Own Year & Aspire for New Year. 1年の終わりに、どのような1年だったかを振り返るとともに、次の1年への希望を集めよう。

## 課題内容（どんな作品か）

(背景)
・年末年始は多くの人と話したくても会えない時もあり、そこで人と人とを繋ぐようなものができたら？と考えた。また、会ったことない多くの人の声を聞くことで、どんな年であったか、来年がどんな年になるのかを考えるマーケターの力にも。皆にとって振り返りができ、来年を楽しみになるきっかけになれば嬉しい。

(内容)
・過ぎ去りゆく年と新たに迎える年を繋ぐアプリ
・年末の振り返りと、年始の願掛けをするきっかけを提供
・さらに他の人のものも共有されるため、俯瞰をして楽しむことができる
・また運営側(例えばマーケター)は生活者の動向や感情を理解し、
　素晴らしい新たな1年をつくるために声を聴くこともできる。
　(簡易インサイト分析あり)
・その投稿が力になる時は、スクショして保存をすることができる(LocalStorageにて)

## 工夫した点・こだわった点
・実際にこの時期にローンチできる企画とデザインを考えつくっていった。
・画像はAIと共に描き、BGMはフリー素材から選び、実現ができた。
・参加者と運営者との両方にとってプラスが生まれる仕組みにした。
・こだわりは3つ
  1. 年末と正月について両方同時に考えられること
  2. 他の皆の声が視覚的に見やすく読むことで参考になること 
  3. ぱっと見参加したくなるデザインやシンプルで直感的なUIUX
・課題としてのアンケートの特徴を使いながらの企画に工夫をした。

## 難しかった点・次回トライしたいこと（又は機能）
